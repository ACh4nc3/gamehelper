
Game Helper Launcher & ESP Loader
这是一个基于 Android Studio 开发的增强型 Shell 执行工具。它专为安卓游戏 C++ 二进制文件（ESP 绘制） 设计，集成了内核驱动刷入、可视化参数配置以及二进制文件的一键启动功能。
🚀 核心功能
 * 内核驱动可视化刷入：摆脱繁琐的命令行，支持通过图形界面（GUI）一键加载或刷入 .ko 等内核驱动模块。
 * C++ 二进制执行器：针对游戏 ESP 绘制项目优化，稳定运行底层 C++ 二进制文件。
 * 可视化操作面板：支持动态调整启动参数，实时查看 Shell 输出日志。
 * 自动化环境配置：自动处理文件权限 (chmod +x) 及运行环境初始化。
 * Root 模式集成：深度适配 Magisk 环境，确保高权限指令的稳定执行。
🛠️ 技术栈
 * 开发环境：Android Studio (Java/Kotlin)
 * 核心逻辑：Linux Shell Script / JNI
 * 底层交互：Root Shell Access (Su)
 * UI 框架：Material Design 3

📥 安装与运行
前提条件
 * 设备已获取 Root 权限（推荐使用 Magisk）。
 * 开启 USB 调试（用于开发调试）。
 * 确保二进制文件与当前设备架构（ARM64/x86）匹配。
编译步骤
 * 克隆仓库：
   git clone https://github.com/ACh4nc3/gamehelper

 * 使用 Android Studio 打开项目。
 * 等待 Gradle 构建完成，点击 Run 部署到设备。
📖 使用指南
 * 准备文件：将你的 C++ 二进制文件（如 esp_draw）放入 APP 指定的私有目录或 SD 卡。
 * 加载驱动：在“驱动管理”页面选择对应的内核模块，点击“刷入”。
 * 配置文件路径：在启动器设置中填入二进制文件的绝对路径。
 * 一键启动：点击主界面的“开启”按钮，APP 将自动申请 Root 权限并执行。
